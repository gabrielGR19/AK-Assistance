# Agenten

Hier werden eigenständige KI-Agenten dokumentiert und gespeichert.

## Was gehört hierher?

Agenten, die eine klar abgegrenzte, eigenständige Aufgabe übernehmen:
- **Marketing-Agent** — Handwerksbetriebe scrapen, Website analysieren, E-Mail generieren
- **Telefon-Agent** — Retell.ai + Claude API Konfiguration
- **Kalender-Agent** — Terminvereinbarung automatisieren

## Dateiformat

Jeder Agent bekommt eine eigene Datei: `<name>-agent.md`

Struktur pro Datei:
```
# Agent: <Name>
## Zweck
## Eingabe
## Ausgabe
## Abhängigkeiten (APIs, Tools)
## Prompt / Systemanweisung
## Beispiel-Run
```
