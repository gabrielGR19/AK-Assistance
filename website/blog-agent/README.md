# Blog-Agent

Platzhalter für den KI-Agenten, der automatisch Blog-Beiträge für die AK-Assistance Website schreibt.

## Geplante Funktion

- Themen automatisch generieren oder entgegennehmen
- Blog-Artikel im passenden Stil verfassen
- Direkt in die Website-Struktur ausgeben

In Vorbereitung.
