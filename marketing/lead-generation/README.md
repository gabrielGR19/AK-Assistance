# Lead-Generierung

Automatisierte Identifikation und Ansprache von Handwerksbetrieben als potenzielle Kunden.

## Geplanter Workflow

1. **Scraping** — Apify findet lokale Betriebe (Google Maps, Branchenverzeichnisse)
2. **Analyse** — Claude bewertet Website-Qualität und Digitalisierungsstand
3. **E-Mail** — Personalisierte Akquise-Mail wird generiert und via n8n versendet
